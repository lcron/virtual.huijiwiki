这是“人物链接”模板。调用它时应该采用如下格式： 
{{人物链接
|名字=
|别名=
|连接=
|头像=
}}
但一般是采用简化后的格式 
{{人物链接|名字}}
=== 效果 === {{人物链接}} 编辑页面以阅读模板文本。 什么鬼，这里为什么会被渲染成代码块? 名字 { "params": { "名字": { "suggested": true, "type":"string" }, "别名": { "suggested": true, "type":"string" }, "连接": { "suggested": true, "type":"string" }, "头像": { "suggested": true, "type":"wiki-file-name" } } }
